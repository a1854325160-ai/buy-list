# Buy List — App UI Kit

## Overview
High-fidelity interactive prototype of the Buy List mobile app. Click-through with real React state.

## Screens
- **Home** — List of shopping lists, category filter, stats card
- **List Detail** — Items with checkboxes, progress, swipe hints
- **Add Item** — Bottom sheet modal with form
- **Profile** — User stats and settings

## Usage
Open `index.html` in a browser. The prototype renders inside an iPhone frame.

## Components
- `AppShell` — Navigation tabs + screen routing
- `HomeScreen` — Dashboard with list cards
- `ListDetailScreen` — Items list with check/uncheck
- `AddItemSheet` — Bottom sheet form overlay
- `ProfileScreen` — User profile and settings
- `ios-frame.jsx` — iPhone bezel (starter component)

## Icons
Uses Lucide Icons via CDN (outline, 1.5px stroke, rounded).

## Fonts
- Plus Jakarta Sans (display/headings)
- DM Sans (body)
- JetBrains Mono (prices/quantities)
